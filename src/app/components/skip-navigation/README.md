# Skip Navigation

## Purpose
Provides keyboard users a quick way to skip repetitive content and move directly to important page regions.

## Accessibility Notes
- Uses visible-on-focus skip links
- Supports keyboard-only navigation
- Moves focus to the target region
- Intended targets should exist on the page and be focusable when activated

## WCAG Criteria
- 2.1.1 Keyboard
- 2.4.1 Bypass Blocks
- 2.4.3 Focus Order
- 2.4.7 Focus Visible
