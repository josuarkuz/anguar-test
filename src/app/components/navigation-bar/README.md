# Navigation Bar

## Purpose
Provides responsive primary navigation with support for desktop and mobile layouts.

## Accessibility Notes
- Uses semantic `nav` landmark
- Mobile menu toggle exposes `aria-expanded`
- Dropdown trigger exposes `aria-haspopup` and `aria-expanded`
- Keyboard support includes Enter, Space, ArrowDown, ArrowUp, Home, End, Escape, and Tab behavior for dropdown items
- Touch targets use a minimum 44px interactive height
