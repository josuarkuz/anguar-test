# Modal Dialog

## Purpose
Provides a focus-trapped dialog for important actions and interruptions.

## Accessibility Notes
- Uses `role="dialog"` and `aria-modal="true"`
- Supports Escape to close
- Traps focus while open
- Restores focus to the previously focused trigger when closed
- Associates title and optional description with the dialog

## WCAG Criteria
- 2.1.1 Keyboard
- 2.1.2 No Keyboard Trap
- 2.4.3 Focus Order
- 2.4.7 Focus Visible
- 4.1.2 Name, Role, Value
