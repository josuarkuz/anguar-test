# Data Table

## Purpose
Provides a semantic, sortable, paginated data table with accessible controls.

## Accessibility Notes
- Uses native table semantics with `table`, `thead`, `tbody`, `th`, and `td`
- Column sort controls are keyboard accessible buttons
- Sort state is exposed with `aria-sort`
- Table has a caption for screen readers
- Pagination status updates are announced with `aria-live`

## WCAG Criteria
- 1.3.1 Info and Relationships
- 2.1.1 Keyboard
- 2.4.3 Focus Order
- 4.1.2 Name, Role, Value
- 4.1.3 Status Messages
